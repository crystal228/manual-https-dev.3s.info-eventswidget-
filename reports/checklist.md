# Checklist

| ID | Проверка | Статус | Комментарий |
| --- | --- | --- | --- |
| CH-001 | Страница `eventswidget` открывается | Pass | Страница загрузилась, основной UI доступен. |
| CH-002 | Генерация preview с настройками по умолчанию | Fail | Preview создается, но ширина не соответствует `width="230"` из embed-кода. |
| CH-003 | Выбор тематики мероприятий | Pass | Выбранные тематики попадают в embed-код как `event_type=1,5`. |
| CH-004 | Выбор стран | Pass | Выбранные страны попадают в embed-код как `event_country=online,ru,us`. |
| CH-005 | Настройка ширины и высоты | Pass | Значения `500x600` попадают в атрибуты iframe. |
| CH-006 | Выбор красной темы | Fail | Для красной темы параметр `theme` отсутствует в embed-коде. |
| CH-007 | Выбор остальных цветовых тем | Pass | Blue, green, turquoise и purple попадают в embed-код. |
| CH-008 | Повторная генерация после изменения настроек | Fail | Preview не всегда обновляет iframe `src`. |
| CH-009 | Проверка embed-кода | Fail | Есть некорректный код для красной темы. |
| CH-010 | Адаптивность desktop/tablet/mobile | Pass | Критичных поломок layout не зафиксировано. |
| CH-011 | Console errors/warnings | Fail | Есть Mixed Content warnings. |
| CH-012 | Failed network requests | Fail | Зафиксирован `net::ERR_ABORTED` для Google Analytics. |
| CH-013 | Edge cases для размеров | Fail | Некорректные значения молча заменяются или ограничиваются без подсказки пользователю. |
| CH-014 | Кнопка копирования embed-кода | Pass | После клика текст меняется на `Скопировано`. |
